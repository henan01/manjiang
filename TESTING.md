# 麻将游戏测试步骤

## 快速测试（单人）

1. **登录**

   - 打开 http://localhost:5173/
   - 输入用户名并登录

2. **创建房间**

   - 点击 "创建房间"
   - 等待进入房间（不再转圈）

3. **开始游戏**

   - 点击 "开始游戏"
   - 查看手牌（14 张）

4. **打牌**

   - 点击选中一张牌
   - 点击 "出牌"

5. **摸牌**

   - 点击 "摸牌"
   - 手牌增加一张

6. **测试刷新**
   - 按 F5 刷新
   - 验证无需重新登录

## 完整测试（4 人）

1. 打开 4 个浏览器窗口
2. 分别登录 4 个不同用户
3. 第一个用户创建房间
4. 其他用户加入房间
5. 房主开始游戏
6. 依次出牌、摸牌
7. 测试碰牌、吃牌、杠牌

## 观战测试（5 人）

1. 4 个玩家已满
2. 第 5 个用户加入 → 自动观战
3. 关闭一个玩家窗口 → 模拟掉线
4. 观战者点击 "上座" → 接管座位

详细测试文档请查看 walkthrough.md
